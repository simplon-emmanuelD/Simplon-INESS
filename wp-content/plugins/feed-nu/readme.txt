== Description ==

Create a native Android app of your blog.
With this plugin you can automatically generate your mobile app ready to deploy on android market or host it by yourself.
No programming skills needed and it's instantly available.

IPhone version comming soon.


== Installation ==

1. Upload `feed-nu` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
